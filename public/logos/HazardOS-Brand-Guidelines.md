### Color Usage Rules

**DO:**
- Use HazardOS Orange for primary actions and calls-to-action
- Use Navy for professional content and body text

## Typography

### Primary Typeface: Geist Mono

**Usage**: Logos, technical UI, code snippets, system messages

**Weights Available**: Regular (400), Bold (700)

**Why**: Monospaced font references terminal interfaces and operating systems. Geometric and readable at all sizes.

**Alternatives**: 
- IBM Plex Mono (free alternative)
- Monaco, Consolas (system monospace)

---

### Secondary Typeface: Inter

**Usage**: Body text, headings, UI labels, marketing materials

**Weights Available**: Regular (400), Medium (500), Semibold (600), Bold (700)

**Why**: Modern, highly readable at all sizes. Designed for screens. Professional without being corporate.

**Alternatives**:
- System fonts: -apple-system, Segoe UI, Roboto

---

### Typography Rules

**Headings:**
- Font: Inter Bold or Semibold
- Color: Navy (#1F2937)
- Size: 24-48px for web, scale appropriately for print

**Body Text:**
- Font: Inter Regular
- Color: Navy (#1F2937) or Gray (#6B7280)
- Size: 16px for web (minimum), 10-12pt for print
- Line height: 1.5-1.6

**Code/Technical:**
- Font: Geist Mono Regular
- Color: Navy (#1F2937)
- Background: Light gray (#F3F4F6)
- Monospace always

**All Caps:**
- Use sparingly (labels, navigation, buttons)
- Never for body text
- Increase letter-spacing (+0.05em) for readability

## Clear Space & Minimum Sizes

### Clear Space

Maintain clear space around all logo variations equal to the height of the octagonal icon.

```
     [  CLEAR SPACE  ]
  ┌─────────────────────┐
  │                     │
  │        [LOGO]       │
  │                     │
  └─────────────────────┘
     [  CLEAR SPACE  ]
```

**Never place:**
- Other logos within clear space
- Text or graphic elements within clear space
- Logo edges touching boundaries

---

### Minimum Sizes

**Digital:**
- **Icon only**: 16px minimum (favicon)
- **Horizontal logo**: 120px width minimum
- **Vertical logo**: 80px width minimum

**Print:**
- **Icon only**: 0.25" (6mm) minimum
- **Horizontal logo**: 1.5" (38mm) width minimum
- **Vertical logo**: 1" (25mm) width minimum

**Below minimum sizes**: Use icon only, no wordmark

## Incorrect Usage

### DON'T:

❌ **Rotate the logo**  
Keep logo horizontal or vertical only. Never at angles.

❌ **Stretch or distort**  
Maintain original proportions. Lock aspect ratio when resizing.

❌ **Change colors**  
Use only approved color variations. No custom colors.

❌ **Add effects**  
No drop shadows, glows, bevels, gradients, or filters.

❌ **Outline the logo**  
Logo already has defined edges. No additional strokes.

❌ **Place on busy backgrounds**  
Use white reverse on dark/busy backgrounds. Ensure contrast.

❌ **Separate elements**  
Keep icon and wordmark together at approved spacing.

❌ **Use low-resolution files**  
Always use vector (SVG) or high-res PNG (300dpi for print).

❌ **Alter letter spacing**  
Wordmark spacing is fixed. Don't adjust.

❌ **Rearrange elements**  
Don't move icon or change icon-to-text relationship.

## File Specifications

### File Types & When to Use

**PNG (Raster)**
- **Use for**: Web, presentations, social media
- **Resolution**: 72-150 dpi for screen, 300 dpi for print
- **Transparency**: Yes (RGBA)
- **Files provided**: All logo variations in multiple sizes

**JPG (Raster)**
- **Use for**: Email attachments, photo integration, file size constraints
- **Resolution**: 300 dpi for print, 72 dpi for web
- **Transparency**: No (white background)
- **Files provided**: Horizontal and vertical, color and b&w

**SVG (Vector)** *(Not currently provided, but recommended for web)*
- **Use for**: Websites, any application needing infinite scaling
- **Resolution**: Infinite (vector)
- **Transparency**: Yes
- **Recommended**: Request SVG versions for web development

**PDF (Vector)**
- **Use for**: Print vendors, professional printing, large format
- **Resolution**: Infinite (vector)
- **Transparency**: Yes
- **Recommended**: Request PDF versions for professional printing

### Website
